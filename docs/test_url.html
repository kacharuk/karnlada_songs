OneDrive URL Tester

How to get working URLs:

  1. Go to onedrive.live.com
  2. Find your file (audio or image)
  3. Right-click → Embed
  4. Copy the URL from src="..." in the iframe code
  5. Paste it below to test

Alternative: Right-click → Download, then check Network tab in browser DevTools for the actual download URL
